MaxLoGs

Maximum of Laplacian of Gaussian filters (imageJ plugin)

What it is?

MaxLoGs is an ImageJ plugin that implement the maximum of Laplacian of Gaussian filter.

How to install ?

ImageJ is required (see http://fiji.sc/, to install FIJI a "batteries-included" distribution of ImageJ). Then just copy the .JAR file in the plugin directory of your ImageJ installation and restart ImageJ.
